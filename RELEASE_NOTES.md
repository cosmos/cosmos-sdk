# Cosmos SDK v0.46.1 Release Notes

This release introduces a few bug fixes and improvements.

Notably, crisis invariant check has been speed up (thanks to a Juno bounty) and the priority mechanism has been improved to depend on gas price.

Please see the [CHANGELOG](https://github.com/cosmos/cosmos-sdk/blob/release/v0.46.x/CHANGELOG.md) for an exhaustive list of changes.
